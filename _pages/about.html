Building a Community

We're aiming to build an open source community in which everyone can contribute and share good resources for learning. Anyone can be a part of our community by joining our organisation on Github and contributing to our projects by writing tech blogs, short tutorials and sharing useful resources. Fork our blog repo and start contributing.

Future Work

Currently we're having a Jekyll blog and we're aiming to build a proper website for this, so that we can have many features and pages like Frequently Occuring Errors(FOE) etc. We'll be migrating as soon as we have sufficient number of people ready to join the project.
